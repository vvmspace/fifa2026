- landings/ - тут хранятся лендинги в сабдиректориях
- landings/boletosfifa.store - [ES] SEO фокус на FIFA 2026, boletos, ...
- landings/copa26.store - [ES] больше на кубок мира

- Лендинги должны отдичаться текстами.
- В title и description должны быть упомянуты  1-3 ближайшие игры
- Тексты должны быть живыми, у поисковых систем не должно быть ассоциаций с ИИ в процессе чтесния текстов.
- Тексты не должны быть перенасыщенными.
- JSON-LD: ссылка должна быть в формате https://[landing.domain]/#[match]`, а если есть страница с билетами, то на страницу с билетами JSON-LD. у лендингов должны быть немного отличаться


Перечисление игр:
- сортировка по дате по возрастанию
- только предстоящие игры
- комментарий должен быть уникален для каждой игры и уникалей для каждого сайта
- если для этой игры есть страница или подраздел, то должны быть ссылки с соответствующим title - всегда проверяй файловую структуру и наличие соответствующих ссылок на главной
- блок в целом не должен создавать проблем для SEO

Страницы игр:
- By demand нужно создавать страницы игр:
- [game]/index.html - главная
- [game][/boletos, /entradas или /tickets, в зависимости от сайта]/index.html - страница с билетами
- [game]/vivo - для онлайн трансляций
- Страница каждой игры должна представлять из себя лендинг.
- Страницы могут быть совсем уникальные.

Обновление игр:
node index.js events.md -> check events.md -> events -> ... -> dates -> ...
После обновления в landings/.../index.html - должно быть обновлено расписание игры с фокусом не ближайшие игры + игры должны выводиться строго после текущей даты/времени.

В таблице игр не должно быть повторяющегося текста.

- верстку: отдельно для каждого лендинга

- у ссылок есть уникальный SEO атрибут title
- соответствие SEO и GEO
- отображение главной: главная - это лицо сайты
- совершенствуй дизайн когда это нужно
- наличие schema.org на странице евента, на главной и там, где упоминается евент
- открывай через браузер, высматривай косяки на скриншотах
- есть флаги стран в карточках
- актуальность sitemap.xml
- на странице игр есть дата игр
- на главке есть: финал, полуфинал, 1/4 финал
